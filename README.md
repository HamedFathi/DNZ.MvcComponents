# DNZ.MvcComponents

Useful UI-Component (HtmlHelper) for ASP.NET Core MVC based-on Popular JavaScript Plugins (Experimental project).

**Includes:**

*   Select2
*   CKEditor
*   DataTables
*   iCheck
*   SweetAlert
*   SweetAlert-Bootstrap
*   Noty
*   Toaster
*   Bootstrap-Dialog
*   BootBox
*   InputMask
*   Bootstrap-TagAutoComplete
*   TypeAhead
*   InputFile
*   JasnyUploader
*   Feulux Wirzard
*   Bootstrap-PersianDateTimePicker
